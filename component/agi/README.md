# AGI组件

AGI（人工智能生成智能）组件基于Spring AI开发，提供基础的AI对话聊天和知识库功能。

## 功能特性

- 智能体对话管理
- 对话历史记录
- 知识库管理（RAG）
- 文档片段管理与向量化

## 核心组件

### 智能体管理
- 智能体创建与配置
- 角色设定和行为规则定义
- 对话历史管理

### 知识库支持
- 原始文档管理（Word、Excel、PDF、HTML等格式）
- 文档切片与向量化存储
- 向量检索与匹配

### 对话管理
- 实时对话流处理
- 对话历史存储与查询
- 多媒体消息支持

## 依赖组件

- global-ai-boot-starter: 全局AI能力支持
- Spring AI相关组件
- 数据库访问组件

## 使用说明

AGI组件提供了基于Spring AI的对话聊天和知识库问答功能，支持对话历史存储和多种消息类型处理。

用户可以通过创建智能体并为其配置角色、知识库等信息，实现定制化的AI对话体验。
