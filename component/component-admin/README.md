# 组件管理组件

随着组件越来越多，组件管理是必不可少的。该组件旨在解决组件之间或多或少涉及的一些功能依赖，
避免添加大量的判断来隔离在组件存在或不存在时的处理。

虽然使用Spring Boot的自动配置能解决一部分问题，但在有些场景下还是难以解决，比如在微服务情况下，
分布式部署时，已经不能通过Spring Boot自动装配来决断整个集群总共使用了哪些组件。
通过提供接口也会给前端提供一些支持，可以获取当前系统可用的组件。
