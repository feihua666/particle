# 角色组件

角色组件管理系统中的角色定义和权限分配，支持角色继承和权限聚合。

本组件认为，有角色一定有用户和功能菜单，所以并没有考虑隔离处理，前端直接引用。

## 使用方法

在项目的 pom.xml 中引入依赖：

```xml
<dependency>
    <groupId>com.particle</groupId>
    <artifactId>role-boot-starter</artifactId>
    <version>${particle.version}</version>
</dependency>
```

## 核心组件

- 角色管理：管理系统中的角色定义
- 用户角色权限：管理用户与角色之间的关联关系
- 角色功能权限：管理角色与功能菜单之间的关联关系
- 角色数据范围权限：管理角色与数据范围之间的关联关系

## 菜单功能

### 权限管理
- 角色管理
  - 查询、创建、删除、修改角色
  
- 用户角色权限
  - 查询、创建、删除、修改用户角色关系
  - 用户分配角色功能
  - 角色分配用户功能
  - 清空角色下的所有用户
  - 清空用户下的所有角色

- 角色功能权限
  - 查询、创建、删除、修改角色功能关系
  - 功能分配角色功能
  - 角色分配功能功能
  - 清空角色下的所有功能
  - 清空功能下的所有角色

- 角色数据范围权限
  - 查询、创建、删除、修改角色数据范围关系
  - 角色分配数据范围功能
  - 数据范围分配角色功能
  - 清空角色下所有数据范围
  - 清空数据范围下所有角色


## 自动配置

组件通过 [RoleAutoConfiguration](role-boot-starter/src/main/java/com/particle/role/RoleAutoConfiguration.java) 实现自动配置，
并扫描 `com.particle.role` 包下的组件，包括：
- Mapper接口：角色、用户角色关系、角色功能关系、角色数据范围关系相关的数据访问层接口
- 应用服务：role-app模块的应用服务类

该组件还集成了Swagger文档功能，提供了后端管理接口文档分组。
