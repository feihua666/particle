# 全局轻量级共享模块

该模块提供全局注解、静态变量等，解决Maven重依赖问题，解耦实现。

## 功能特性

1. **全局注解**：提供项目范围内通用的注解定义
2. **静态变量**：定义项目范围内的公共静态常量
3. **轻量级设计**：最小化依赖，避免循环依赖问题
4. **解耦支持**：帮助各模块之间解耦，降低模块间耦合度

## 使用方法

### 添加依赖

```xml
<dependency>
    <groupId>com.particle</groupId>
    <artifactId>global-common</artifactId>
</dependency>
```

### 配置选项

该模块通常不需要特殊配置。

### 核心组件

- 全局注解类：项目范围内的通用注解
- 常量类：公共静态常量定义
- 工具类：轻量级工具方法

## 依赖组件

- 无强制依赖
