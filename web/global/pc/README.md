# 前端 PC 通用组件

前端PC通用的应该放在该目录下，这些组件与具体业务无关，属于技术层面的通用能力封装。

## 目录结构

- [common](./common/README.md) - 通用工具组件
- [element-plus](./element-plus/README.md) - Element Plus组件封装

## 核心组件

PC端通用组件是 Particle 前端项目在PC端的基础技术支撑层，提供各种与业务无关的技术能力封装。这些组件可以在整个PC端系统中被复用，确保技术栈的一致性和系统的稳定性。

## 设计理念

PC端通用组件的设计遵循以下原则：

1. **高内聚低耦合**: 每个组件专注于特定的技术功能，减少组件间的依赖
2. **开箱即用**: 大多数组件提供合理的默认配置，降低使用门槛
3. **可扩展性**: 组件设计考虑了可扩展性，允许根据具体需求进行定制
4. **统一规范**: 提供系统级的技术规范和标准，确保整个系统的一致性
