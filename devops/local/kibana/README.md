# 本地kibana服务器
kibana 为es的一个可视化界面
|功能| 命令                 | 备注                                                                                    |
| --- |--------------------|---------------------------------------------------------------------------------------|
|启动kibana| `./start-kibana.sh` | kibana browser访问：[http://localhost:7474/](http://localhost:7474/) 默认用户/密码为：kibana/kibana |
|停止kibana| `./stop-kibana.sh`  | 将清空所有network和volume                                                                   |
