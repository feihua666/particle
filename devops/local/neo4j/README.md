# 本地neo4j服务器

|功能| 命令                 | 备注                                                                                    |
| --- |--------------------|---------------------------------------------------------------------------------------|
|启动neo4j| `./start-neo4j.sh` | neo4j browser访问：[http://localhost:7474/](http://localhost:7474/) 默认用户/密码为：neo4j/neo4j |
|停止neo4j| `./stop-neo4j.sh`  | 将清空所有network和volume                                                                   |
