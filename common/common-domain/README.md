# 基础模型实体模块

通用模型实体模块，主要包括基础的模型基类和领域驱动设计相关组件。

## 功能特性

- 领域驱动设计基础模型定义
- 实体、值对象基类实现
- 领域服务接口定义
- 仓储网关接口定义
- 领域事件基类定义

## 核心组件

### 基础模型类
- `AggreateRoot`: 聚合根基类
- `ValueObjRoot`: 值对象基类
- `Id`: ID 基类

### 领域服务组件
- `IBaseDomainService`: 基础领域服务接口
- `IGateway`: 网关标记接口
- `IBaseGateway`: 基础网关接口

### 领域事件
- `DomainEvent`: 领域事件基类
- `TemplatingDomainMessageEvent`: 模板化领域消息事件

## 依赖组件

- 无强依赖，主要依赖 JDK 基础类库和部分工具类库（如 Guava）

## 使用说明

领域层模块是系统的核心业务逻辑层，包含实体、值对象、领域服务和仓储接口等。该模块为具体的业务模块提供了领域驱动设计的基础支撑。
